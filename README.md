# 🚀 AI Multiverse Chatbot (Custom Persona System)

이 프로젝트는 Spring AI와 H2 Database를 결합하여, 사용자가 자신만의 캐릭터(페르소나)를 생성하고 이를 영구적으로 저장 및 관리할 수 있는 지능형 챗봇 플랫폼입니다.

## 🌟 주요 기능 (Key Features)

* **다중 세계관 페르소나**: 기본 제공되는 셜록 홈즈, 고든 램지 외에도 사용자가 원하는 성격을 가진 캐릭터를 무한대로 생성할 수 있습니다.
* **영구 기억 저장소 (Persistent Memory)**: H2 Database를 사용하여 서버를 재시작해도 생성한 캐릭터 정보와 대화 기록이 소실되지 않고 안전하게 보존됩니다.
* **스마트 지침 자동 주입**: 커스텀 캐릭터 생성 시, 사용자의 이름을 기억하고 불러주는 '기억 가이드' 프롬프트가 자동으로 결합되어 더 친근한 상호작용이 가능합니다.
* **사용자 정의 아바타 크롭**: 캐릭터 생성 시 원하는 이미지를 업로드하고 원하는 구도로 직접 크롭하여 캐릭터 프로필로 활용할 수 있습니다.
* **캐릭터 관리 시스템**: 원치 않는 캐릭터를 즉시 삭제할 수 있는 관리 기능을 지원합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Backend**: Spring Boot, Spring AI, Spring Data JPA
* **Database**: H2 Database (File Mode for Persistence)
* **Frontend**: HTML5, JavaScript, Cropper.js (이미지 편집)
* **Interface**: Glassmorphism 디자인 적용 (현대적이고 감각적인 UI)

---

### 💡 사용자 매뉴얼

1. **캐릭터 생성**: `+ 캐릭터 추가하기` 버튼을 누르고 이름, 한 줄 소개, 페르소나 성격을 입력하세요.
2. **프로필 설정**: 이미지를 선택한 뒤 자유롭게 크롭하여 생성 완료를 누르면 사이드바에 즉시 추가됩니다.
3. **지능형 대화**: 캐릭터를 클릭하여 대화를 시작하세요. "내 이름은 비씨디야"라고 말하면 AI가 이를 기억하고 다음 대화부터 친근하게 불러줍니다.
4. **영구 관리**: 생성된 캐릭터는 DB에 저장되므로 브라우저를 새로고침하거나 서버를 재시작해도 사라지지 않습니다.

---

* **DB 확인**: `http://localhost:8080/h2-console`에 접속하여 `CUSTOM_CHARACTER` 테이블을 통해 저장된 페르소나 데이터를 직접 확인/수정할 수 있습니다.
